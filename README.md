## Installation

```
$ cd ~
$ git clone https://github.com/ystein/.files
$ cd .files
$ make
File already exists: /home/ystein/.zshenv, what do you want to do?
[s]kip, [S]kip all, [o]verwrite, [O]verwrite all, [b]ackup, [B]ackup all
$ B
  ```

## Removal

```
$ make clean
```
